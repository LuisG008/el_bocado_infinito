<h3>Configuracion Aplicación</h3>
<p>Se debe crear el .env.local, agregar y configurar las siguientes variables
</p>
<ul>
    <li>APP_ENV=dev</li>
    <li>APP_DATABASE_HOST=</li>
    <li>APP_DATABASE_USER=</li>
    <li>APP_DATABASE_PASS=</li>
    <li>APP_DATABASE_DB=bocado_infinito</li>
    <li>APP_DATABASE_PORT=3306</li>
</ul>
